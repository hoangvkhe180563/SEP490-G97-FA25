Generate DB vào đây.
Chuột phải project mở terminal. (KHÔNG MỞ TRONG FOLDER DATA)
Câu lệnh để generate:

dotnet ef dbcontext scaffold "server=localhost;database=studyhub;user=root;password=yourpassword" Pomelo.EntityFrameworkCore.MySql -o Data -c AppDbContext -f

Trong đó: -c: tên file dbcontext, -f: nếu có sửa DB thì khi scaffold bắt nó override
Sau khi scaffold xong, bỏ OnConfiguring và bỏ comment ở file DependencyInjection.cs
